# AgentVault Memory

## 简介

开源项目（2026年4月发布），提供跨 AI 工具（Claude Code、Cursor 等）的统一记忆层，直接读取本地历史记录，输出为 Obsidian Markdown 格式。

## 关键信息

- 发布: 2026年4月
- 定位: 跨 AI 工具的统一记忆中间件
- 输出: Obsidian Markdown
- 在 AI 记忆中枢架构中处于「统一记忆层」
