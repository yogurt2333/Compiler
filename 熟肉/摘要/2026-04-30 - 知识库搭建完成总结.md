# 知识库搭建完成总结

## 核心内容

本次对话是知识库搭建完成后的总结回顾，梳理了完整的知识编译流水线和工具生态。

1. **知识流水线**：AI对话/文章 -> Web Clipper 剪藏 -> 生肉/对话/ -> Claude Code 编译 -> 熟肉/(概念/摘要/综合/)
2. **工具清单**：Obsidian(底座)、Web Clipper(摄入)、Claude Code(编译)、Claudian(内联AI)、DeepSeek API(推理)
3. **日常流程**：每天 5 秒剪藏 + 每周 10 分钟 AI 批量编译 + 随时搜索和内联编辑

## 踩坑汇总

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| Web Clipper 保存后为空 | 未配置 `{{content}}` | 在「笔记内容」填 `{{content}}` |
| 文件名是乱码时间戳 | 用了 `{{date}}` | 改用 `{{date\|date:"YYYY-MM-DD"}}` |
| 模板报错 `#selector` | Web Clipper 不支持 | 删掉那一行 |
| 只有最近几条对话 | 动态加载未滚动 | 手动滚动到底部再全选复制 |
| Claudian 找不到 claude | 默认找 .exe | 手动指定 `.exe` 路径 |
| CLAUDE.md 未加载 | cwd 在子目录 | 确保在根目录运行 `claude` |
