# 2026-05-04 Claude Code / Hermes / Obsidian 三端数据打通方案

## 核心内容

将 Claude Code、Hermes Agent、Obsidian 三者的历史记录统一转换为 Markdown 存入 Obsidian，再通过本地数据库做语义检索。

## 打通路径

**第一步: 导出 Claude Code 历史**
- 数据位置: `~/.claude/projects/` 下的 `.jsonl` 文件
- 推荐工具: `claude-conversation-extractor` (Python) 或 `cclog` (Go)
- 导出格式: Markdown，直接存入 Obsidian 仓库

**第二步: 定位 Hermes 数据**
- 会话 JSON: `~/.hermes/sessions/`
- 核心记忆: `~/.hermes/memories/`（USER.md、MEMORY.md）
- 元数据 SQLite: `~/.hermes/state.db`

**第三步: 检索 Obsidian**
- 推荐 ripgrep (`rg`) 全文搜索
- 可构建本地向量库做语义检索

**第四步 (进阶): 第三方记忆框架**
- MemVerse: 多模态支持 + 记忆蒸馏技术

## 隐私
整套流程 100% 离线，所有数据均存储在本地。
