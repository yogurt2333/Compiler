## 🔁 日常使用流程

### 每天（5秒）
1. 在Claude/ChatGPT完成对话
2. 滚动到底部加载全部 → `Ctrl+A` → `Ctrl+C`
3. 点击浏览器Web Clipper图标 → “添加到Obsidian”
4. 文件自动保存到 `生肉/对话/2026-04-30 - 主题.md`

### 每周（10分钟）
在终端 `C:\AI\Obsidian\Compiler` 运行：
```
请扫描 生肉/对话/ 下本周新增的文件，按照CLAUDE.md规则编译到 熟成/ 目录下
```
AI会：
- 提取核心概念 → `熟成/概念/`
- 生成摘要 → `熟成/摘要/`
- 识别实体 → `熟成/实体/`
- 发现矛盾 → 用⚠️标注
- 更新 `log.md` 和 `index.md`

### 随时（需要时）
- **搜索**：`Ctrl+Shift+F` 搜关键词，优先看 `熟成/概念/`
- **内联编辑**：选中文字 → Claudian侧边栏 → “帮我提炼/润色/关联”
- **健康检查**：`请扫描熟成/概念/，找出定义模糊或过时的内容`

## ⚠️ 踩过的坑 & 解决方案（永久避雷）

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| Web Clipper保存后是空的 | 未配置`{{content}}` | 在「笔记内容」填`{{content}}` |
| 文件名是乱码时间戳 | 用了`{{date}}` | 改用`{{date\|date:"YYYY-MM-DD"}}` |
| 模板报错`#selector` | Web Clipper不支持 | 删掉那一行 |
| 只有最近几条对话 | 动态加载未滚动 | 手动滚动到底部再全选复制 |
| Claudian找不到claude | 默认找.exe，实际是.ps1 | 手动指定`.exe`路径 |
| CLAUDE.md未加载 | cwd在子目录 | 确保在根目录运行`claude` |

## 🎯 三个最高杠杆的动作（现在就做）

1. **给 `熟成/概念/` 每个文件加“一句话定义”**  
   → 以后搜索扫一眼就懂，不用点开

2. **把最常用的5个概念 `Ctrl+B` 加入书签**  
   → 每天省10次点击

3. **在 `熟成/综合/` 建一个 `README.md`**，写：  
   > “我为什么建这个知识库？我最关心的3个问题是：1. AI Agent架构 2. RAG优化 3. 知识管理方法”  
   → 防止知识库变成“别人的知识库”，保持自己的方向感
