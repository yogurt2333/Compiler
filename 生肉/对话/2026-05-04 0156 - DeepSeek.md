## 🚀 二、Hermes Agent 安装与配置

### 1. 安装
```bash
curl -fsSL https://res1.hermesagent.org.cn/install.sh | bash
source ~/.bashrc
hermes --version   # 验证安装
```

### 2. 配置大模型（DeepSeek V4 Flash）
```bash
hermes config set model.provider deepseek
hermes config set model.base_url https://api.deepseek.com
hermes config set model.api_key sk-你的Key
hermes config set model.default deepseek-v4-flash
```

> **选 Flash 的原因**：响应更快（毫秒级）、价格更低（输出 2元/百万 token），适合日常对话和任务规划。

### 3. 验证模型连通性
```bash
hermes doctor   # 应显示 ✓ DeepSeek
hermes          # 测试对话，输入“你好”
```

## 📱 三、微信接入（ClawBot）

### 配置命令
```bash
hermes gateway setup
```

| 配置项 | 选择 |
|--------|------|
| 平台 | Weixin |
| 私聊授权 | Use DM pairing approval（个人使用可改为 `allow_all_users true`） |
| 群聊 | Disable（官方不支持群聊） |
| 后台服务 | User service（开机自启） |

### 扫码与配对
- 终端输出二维码链接 → 浏览器打开 → **微信小号**扫码
- 微信中自动添加「**微信 ClawBot**」
- 首次需批准：`hermes pairing approve weixin [CODE]`

### 后台保活
```bash
sudo loginctl enable-linger $USER   # 用户服务开机自启
```

### ClawBot 的 Markdown 支持
✅ ClawBot 聊天界面**支持 Markdown 渲染**（标题、列表、代码块、表格等），普通聊天/群聊不支持。

## ⚠️ 八、踩坑与解决记录

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| `hermes: command not found` | PATH 未更新 | `source ~/.bashrc` |
| API 401 错误 | Key 无效 / 账户余额不足 | 去 DeepSeek 控制台重新生成 Key 并充值 |
| 微信无回复 | 未批准配对 | `hermes pairing approve` |
| Base URL 填错 | 误把 API Key 填到 URL | 正确：`https://api.deepseek.com` |
| Browser 收到连接拒绝 | 安全组已通但网关未启动 | 启动 `hermes gateway` |
