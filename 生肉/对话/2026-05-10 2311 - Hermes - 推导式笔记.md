# Python推导式

## 定义
推导式（Comprehension）是 Python 将 for 循环、变换、过滤合并为一行表达式的语法糖。

相当于 JS 的 `filter + map` 链式调用合体，语法更接近 SQL 的 SELECT-WHERE 风格。

## 核心要点

### 三种推导式

```python
# 列表推导式  [表达式 for 变量 in 可迭代对象 if 条件]
[x * 2 for x in range(10) if x % 2 == 0]

# 字典推导式
{i: i**2 for i in range(5)}

# 集合推导式（自动去重）
{x * 2 for x in [1, 2, 2, 3, 3, 3]}
```

### 相比 for 循环的优点
1. 不用手动管理 index，零串位风险
2. 代码更短、更声明式
3. 性能略优于 for 循环（C 层实现）

### 注意点
- 只生成新的序列，不修改原数据
- 嵌套推导式可读性差，超过两层建议展开为 for 循环
- 推导式创建的是新容器（浅拷贝），嵌套对象需要配合 `copy.deepcopy`

## 常见混淆
推导式 vs 深/浅拷贝 —— 完全不同的维度：
- 推导式 = 怎么生成列表（filter + map 一行写）
- 深/浅拷贝 = 引用类型复制策略（复制引用还是复制值）
