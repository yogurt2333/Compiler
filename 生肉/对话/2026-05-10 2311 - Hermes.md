# 每日一题 2026-05-10：用 index 遍历删元素会串位

## 核心问题
用 index 遍历 + 原地修改 = 串位

### 第一段：遍历删元素
```python
for i in range(len(nums)):
    if nums[i] % 2 == 0:
        nums.pop(i)   # 删除后元素左移，i跳过下一个 -> 串位
```

### 第二段：变体——合并有序数组（LeetCode 88）
```python
# 从前往后填 -> 还没处理的值被覆盖
# 从后往前填 -> 三个指针倒着走，安全
```

### 第三段：统一解法
| 场景 | 解法 |
|:--|:--|
| 遍历删元素 | 倒序遍历 `for i in range(len-1, -1, -1)` |
| 合并有序数组 | 从后往前填 |
| 最省心 | 列表推导式，不动 index，零串位风险 |

**口诀**：原地改数组，先想倒着走；能写推导式，不碰 index 手

---

## 推导式（Comprehension）

本质：for 循环的语法糖，≈ JS 的 `filter + map` 一行写

```python
# 列表推导式
[x * 2 for x in range(10) if x % 2 == 0]

# 字典推导式
{i: i**2 for i in range(5)}

# 集合推导式
{x * 2 for x in [1, 2, 2, 3, 3, 3]}
```

语法：`[表达式 for 变量 in 可迭代对象 if 条件]`

## 常见混淆澄清

| 概念 | 解决什么问题 |
|:--|:--|
| 推导式 | 怎么优雅地生成新列表（map+filter） |
| 深/浅拷贝 | 引用类型复制时，复制引用还是复制值 |

两者不同维度，可以组合用，但概念独立。
